Descrizione
----------
Progetto front-end per LG-Shoes. Demo disponibile su GitHub Pages.

Anteprima
--------
Clicca sul pulsante qui sopra per vedere il sito in produzione.

Esecuzione locale
-----------------
1. git clone https://github.com/cammo22/LG-Shoes.git
2. cd LG-Shoes
3. Apri index.html nel browser o esegui un server locale:
   - python -m http.server 8000
   - npx http-server
